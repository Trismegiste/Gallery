# A personal gallery for my photo

## Guidances
 * Mobile first
 * As [KISS][5] as possible
 * As light as possible
 * Because Picasa and Flickr are feature creeps

## Technologies
 * [Silex with Trait][1]
 * [Twig][2]
 * [PureCSS][3]
 * [Photoswipe][4]
